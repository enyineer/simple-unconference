Session assignment priority: moderators and owners can optionally set a priority (High / Normal / Low) on a session. High-priority sessions win a room ahead of star count and are filled with attendees first (low-priority last) in both the per-slot and whole-agenda assignment - useful for sessions that need many participants. Priority only steers people between sessions they starred and never overrides manual placements, pinned rooms, or capacity. Visible as a badge on session cards and in the slot sheet; documented in the assignment rules dialog.
